Git and GitHub Guide

What is Git?

Git is a version control system that tracks changes in your files, allowing collaboration and easy management of your codebase.

What is GitHub?

GitHub is a platform for hosting and managing Git repositories online, enabling teams to collaborate on projects.

Setup Git

  1. Install Git: Download Git from git-scm.com and follow the installation instructions.
  2. Configure Git: 
    git config --global user.name "Your Name"
git config --global user.email "youremail@example.com"
  3. Check Configuration: 
    git config --list

Basic Git Commands

Initialize a Repository

Start tracking files in a folder:

git init

Clone a Repository

Copy an existing repository from GitHub:

git clone <repository-url>

Check Repository Status

See the current status of files:

git status

Add Files to Staging

Stage specific files for the next commit:

git add <file-name>

Stage all files:

git add .

Commit Changes

Save staged changes to the repository:

git commit -m "Your commit message"

View Commit History

See the log of commits:

git log

Connect to Remote Repository

Add a remote repository URL:

git remote add origin <repository-url>

Push Changes to GitHub

Send changes to the remote repository:

git push origin <branch-name>

Pull Changes from GitHub

Download changes from the remote repository:

git pull origin <branch-name>

Branching and Merging

Create a New Branch

git branch <branch-name>

Switch to a Branch

git checkout <branch-name>

Create and Switch to a New Branch

git checkout -b <branch-name>

Merge Branches

git merge <branch-name>

Undo Changes

Unstage Files

git reset <file-name>

Revert to a Previous Commit

git revert <commit-id>

GitHub Commands

Fork a Repository

Click "Fork" on the repository page on GitHub to create a copy under your account.

Open a Pull Request

  1. Push your branch to GitHub.
  2. Go to the repository on GitHub.
  3. Click "Compare & Pull Request".
  4. Write a description and submit the pull request.

Helpful Commands

View Current Branch

git branch

Delete a Branch

git branch -d <branch-name>

Force Push

Use with caution; it overwrites remote history:

git push origin <branch-name> --force

For more information, visit the official Git documentation.
